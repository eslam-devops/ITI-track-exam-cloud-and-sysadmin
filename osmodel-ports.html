موديل OSI
Q1: Which OSI layer is responsible for data encryption?
A) Network
B) Presentation
C) Session
D) Application
عرض الشرح
الـ Presentation Layer هي اللي بتعمل Encryption و Encoding/Decoding
Q2: At which OSI layer does TCP operate?
A) Network
B) Transport
C) Session
D) Data Link
عرض الشرح
TCP و UDP شغالين في الـ Transport Layer
Q3: Which OSI layer handles MAC addresses?
A) Data Link
B) Physical
C) Network
D) Application
عرض الشرح
MAC Address بيكون في الـ Data Link Layer
Q4: Where does IP addressing occur in OSI model?
A) Data Link
B) Physical
C) Network
D) Transport
عرض الشرح
الـ IP Address موجود في Network Layer
Q5: Which OSI layer is responsible for establishing and terminating sessions?
A) Presentation
B) Session
C) Application
D) Network
عرض الشرح
Session Layer مسؤولة عن إدارة وإنهاء الـ Sessions
Q6: Which OSI layer transmits raw bits over a physical medium?
A) Data Link
B) Network
C) Physical
D) Transport
عرض الشرح
Physical Layer هي اللي بتمرر الإشارات الكهربائية والكابلات
Q7: Where does DNS primarily operate in OSI model?
A) Application
B) Network
C) Transport
D) Session
عرض الشرح

Q8: In which layer would you find HTTP?
A) Transport
B) Application
C) Network
D) Data Link
عرض الشرح
HTTP بيشتغل في الـ Application Layer
Q9: Which layer is responsible for multiplexing using ports?
A) Transport
B) Network
C) Data Link
D) Session
عرض الشرح

Q10: Encoding happens in which OSI layer?

B) Presentation
C) Network
D) Application
عرض الشرح
