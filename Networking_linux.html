Networking and Linux Administration Quiz
1. Which layer of the OSI model is responsible for the physical transmission of data?
Data Link
Physical
Network
Transport
View Explanation
الإجابة الصحيحة: Physical. الطبقة المادية (Physical Layer) هي المسؤولة عن نقل البيانات فعليًا عبر الوسائط المادية مثل الكابلات والألياف البصرية.
2. What is the primary function of the Data Link layer?
Routing
Error detection and MAC addressing
End-to-end communication
Data encryption
View Explanation
الإجابة الصحيحة: Error detection and MAC addressing. الطبقة الثانية (Data Link Layer) مسؤولة عن الكشف عن الأخطاء ومعالجتها، بالإضافة إلى استخدام عناوين MAC للتواصل داخل الشبكة المحلية.
3. Which layer of the OSI model handles end-to-end connections and data transmission reliability?
Session
Transport
Presentation
Application
View Explanation
الإجابة الصحيحة: Transport. الطبقة الرابعة (Transport Layer) مسؤولة عن ضمان توصيل البيانات بشكل موثوق بين الأجهزة، باستخدام بروتوكولات مثل TCP.
4. What does the Presentation layer do in the OSI model?
Manages device addressing
Ensures data is properly formatted and encrypted
Handles the physical transmission of data
Manages user interfaces
View Explanation
الإجابة الصحيحة: Ensures data is properly formatted and encrypted. الطبقة السادسة (Presentation Layer) مسؤولة عن تنسيق البيانات وترميزها وضغطها لضمان أنها يمكن أن تفهمها الطبقة التطبيقية.
5. Which OSI layer is responsible for establishing, managing, and terminating sessions between applications?
Transport
Session
Presentation
Application
View Explanation
الإجابة الصحيحة: Session. الطبقة الخامسة (Session Layer) مسؤولة عن إدارة الجلسات بين التطبيقات، بما في ذلك إنشاء الجلسات وصيانتها وإنهائها.

الإجابة الصحيحة: Public IP. عنوان IP العام (Public IP) يستخدم للتواصل خارج الشبكة المحلية، مثل الوصول إلى الإنترنت.
10. What is the maximum number of hosts in a subnet with a /24 subnet mask?
254
255
256
253
View Explanation
الإجابة الصحيحة: 254. في قناع الشبكة /24، يتم استخدام 8 بتات للمضيفين، مما يسمح بـ 2^8 - 2 = 254 عنوانًا صالحًا للمضيفين.
11. What does a subnet mask of 255.255.255.0 indicate?

View Explanation
الإجابة الصحيحة: 24 bits for the network and 8 bits for hosts. قناع الشبكة 255.255.255.0 يشير إلى أن 24 بت الأولى مخصصة للشبكة و8 بت الأخيرة مخصصة للمضيفين.
